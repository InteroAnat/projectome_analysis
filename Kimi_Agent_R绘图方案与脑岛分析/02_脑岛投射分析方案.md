## 1. 脑岛解剖学基础

### 1.1 Evrard 2012/2014分区体系

```
                    前岛叶 (Anterior)                    后岛叶 (Posterior)
                         │                                      │
    ┌────────────────────┼────────────────────┐    ┌───────────┴───────────┐
    │                    │                    │    │                       │
   FI                  Iai                  Ial   Id                     Ig
 (Frontoinsular)   (Intermediate)       (Lateral)  (Dysgranular)      (Granular)
    │                    │                    │    │                       │
 含VENs               含VENs              无VENs  过渡区              初级内感受区
    │                    │                    │    │                       │
 自我感知           情绪整合            认知控制   多模态整合          体感/内脏感觉
```

### 1.2 关键区域定义

| 缩写 | 全称 | 细胞构筑 | 功能特征 |
|-----|------|---------|---------|
| FI | Frontoinsular cortex | Agranular | VENs密集，自我感知 |
| Iai | Insula agranular anterior, intermediate | Agranular | VENs存在，情绪整合 |
| Ial | Insula agranular anterior, lateral | Agranular | 认知控制 |
| Iam | Insula agranular anterior, medial | Agranular | 自主神经调节 |
| Iapm | Insula agranular posterior, medial | Agranular | 过渡区 |
| Iapl | Insula agranular posterior, lateral | Agranular | 过渡区 |
| Id | Insula dysgranular | Dysgranular | 多模态整合 |
| Ig | Insula granular | Granular | 初级内感受皮层 |

### 1.3 左右不对称性

**左岛叶**（语言优势半球）：
- 语言相关内感受
- 时间感知
- 认知控制

**右岛叶**（非语言半球）：
- 情绪处理
- 自主神经调节
- 疼痛感知

## 2. Excel数据结构设计

### 2.1 Sheet 1: Projection_Data（投射矩阵）

**结构**：
```
| Neuron_ID | Left_Ig | Left_Id | Left_Iai | Left_Ial | ... | Right_Ig | Right_Id | ... |
|-----------|---------|---------|----------|----------|-----|----------|----------|-----|
| Neuron_1  | 0.5     | 1.2     | 3.4      | 0.8      | ... | 0.3      | 0.9      | ... |
| Neuron_2  | 0.2     | 0.8     | 2.1      | 1.5      | ... | 0.6      | 1.2      | ... |
```

**列命名规范**：
- 格式：`[Hemisphere]_[Region]`
- 示例：`Left_Ig`, `Right_Iai`, `Left_FI`

**数值说明**：
- 投射到脑岛的axon length（微米）
- 或log转换后的值
- 0表示无投射

### 2.2 Sheet 2: Metadata（神经元元数据）

**必需列**：
```
| Neuron_ID | Soma_Hemisphere | Soma_Region | Soma_Subregion | Neuron_Type | Total_Axon_Length | N_Branches |
|-----------|-----------------|-------------|----------------|-------------|-------------------|------------|
| Neuron_1  | Left            | Insula      | Iai            | IT          | 15000             | 45         |
| Neuron_2  | Right           | PFC         | Area46         | PT          | 23000             | 62         |
```

**列说明**：
- `Neuron_ID`：唯一标识符
- `Soma_Hemisphere`：胞体所在半球（Left/Right）
- `Soma_Region`：胞体所在大区域（Insula/PFC/ACC等）
- `Soma_Subregion`：胞体亚区（Iai/Ial/Id/Ig/Area46等）
- `Neuron_Type`：神经元类型（IT/PT/CT/ITc/ITs等）
- `Total_Axon_Length`：总轴突长度
- `N_Branches`：分支数量

### 2.3 Sheet 3: Region_Info（脑区信息）

**结构**：
```
| Region_Name | Hemisphere | Major_Division | Cytoarchitecture | Functional_System | Is_Insula |
|-------------|------------|----------------|------------------|-------------------|-----------|
| Left_Ig     | Left       | Posterior      | Granular         | Interoceptive     | TRUE      |
| Left_Iai    | Left       | Anterior       | Agranular        | Affective         | TRUE      |
| Right_FI    | Right      | Anterior       | Agranular        | Self-awareness    | TRUE      |
| Left_ACC    | Left       | Medial         | Agranular        | Cognitive         | FALSE     |
```

**关键字段**：
- `Is_Insula`：标记是否为脑岛区域（便于筛选）
- `Cytoarchitecture`：细胞构筑类型
- `Functional_System`：功能系统分类

### 2.4 Sheet 4: Insula_Subdivision（脑岛细分信息）【可选】

**针对脑岛的特殊信息**：
```
| Subregion | VEN_Density | Has_Fork_Cells | Anterior_Posterior | Dorsal_Ventral |
|-----------|-------------|----------------|-------------------|----------------|
| FI        | High        | Yes            | Anterior          | Ventral        |
| Iai       | Medium      | Yes            | Anterior          | Intermediate   |
| Ig        | None        | No             | Posterior         | Dorsal         |
```

insula_neurons <- metadata %>% 
    filter(Soma_Region == "Insula")

3. **亚区特异性**：FI/Iai/Ial/Iam的投射差异

## 5. 从Target Region出发的分析

### 5.1 投射到脑岛的神经元

**筛选条件**：
```r
# 投射到任何脑岛区域
projects_to_insula <- projection_data %>%
    filter(if_any(starts_with("Left_I") | starts_with("Right_I"), ~ . > 0))
```

**分析内容**：
1. **来源区域分布**：哪些脑区的神经元投射到脑岛？
2. **投射强度分布**：投射强度的统计分布
3. **神经元类型**：投射到脑岛的神经元类型（IT/PT/CT）

### 5.2 特定脑岛亚区的输入

**示例：投射到FI的神经元**
```r
projects_to_FI <- projection_data %>%
    filter(Left_FI > 0 | Right_FI > 0)
```

**分析**：
- 来源区域的层次聚类
- 投射模式的相似性
- 与VENs的关系

### 5.3 投射模式的聚类

**目标**：识别投射到脑岛的不同模式

**方法**：
1. 提取投射到脑岛的神经元
2. 对投射向量进行层次聚类
3. 识别投射模式亚型
4. 关联soma位置

**比较维度**：
| 维度 | 左岛叶 | 右岛叶 |
|-----|--------|--------|
| 投射目标 | 语言区、认知区 | 情绪区、自主神经区 |

| 连接模式 | 网络特征 | 网络特征 |

### 6.2 不对称性指数

**计算方法**：
```r
# 不对称性指数 = (Left - Right) / (Left + Right)
asymmetry_index <- (left_projection - right_projection) / 
                   (left_projection + right_projection)
```

**可视化**：
- 不对称性热图
- 显著差异的统计检验
