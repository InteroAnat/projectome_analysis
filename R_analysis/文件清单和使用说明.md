## 数据准备检查清单

### ✅ Projection_Data Sheet
- [ ] 第一列是Subtype名称
- [ ] 后续列是脑区名称
- [ ] 数值是投射强度（log轴突长度）
- [ ] 没有空值或NA

### ✅ Metadata Sheet
- [ ] 包含Subtype, Type, Soma_Region列
- [ ] Subtype名称与Projection_Data匹配
- [ ] Type值为: ITc/ITs/CT/PT/ITi
- [ ] 可选: Total_Axon_Length, N_Neurons

### ✅ Region_Info Sheet
- [ ] 包含Region, Hemisphere列
- [ ] Region名称与Projection_Data列名匹配
- [ ] Hemisphere值为: Ipsilateral/Contralateral

## 常见问题

### Q1: 应该选择R还是Python版本？

| 场景 | 推荐 |
|------|------|
| 需要发表高质量论文图 | R (ComplexHeatmap) |
| 快速查看数据 | Python (seaborn) |
| 已有Python工作流 | Python |
| 需要复杂注释条 | R |
| 不想安装R环境 | Python |

### Q2: 如何调整热图大小？

**R语言:**
```r
pdf("output.pdf", width = 16, height = 12)  # 修改这里
```

**Python:**
```python
fig = plt.figure(figsize=(18, 14))  # 修改这里
```

### Q3: 如何修改颜色？

**R语言:**
```r
col_fun <- colorRamp2(c(0, 2, 5), c("white", "yellow", "red"))
```

**Python:**
```python
plt.imshow(data, cmap='Reds')  # 修改cmap参数
```

### Q4: 数据太大怎么办？

1. 只显示部分亚型
2. 合并相似的脑区
3. 使用降采样
4. 关闭行列标签显示
