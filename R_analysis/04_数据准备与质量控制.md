## 1. Excel文件结构

### 1.1 必需Sheet

你的Excel文件必须包含以下3个Sheet：

```
📁 neuron_projection_data.xlsx
├── 📄 Sheet 1: Projection_Data
├── 📄 Sheet 2: Metadata
└── 📄 Sheet 3: Region_Info
```

### 1.2 Sheet 1: Projection_Data

**用途**：存储每个神经元到各脑区的投射强度

**结构示例**：

| Neuron_ID | Left_Ig | Left_Id | Left_Iai | Left_FI | Right_Ig | Right_Id | ... |
|-----------|---------|---------|----------|---------|----------|----------|-----|
| Neuron_1  | 0.5     | 1.2     | 3.4      | 0.8     | 0.3      | 0.9      | ... |
| Neuron_2  | 0.2     | 0.8     | 2.1      | 1.5     | 0.6      | 1.2      | ... |
| Neuron_3  | 1.0     | 0.5     | 0.3      | 0.2     | 0.8      | 0.4      | ... |

**列说明**：
- **第1列**：`Neuron_ID` - 神经元的唯一标识符
- **后续列**：各脑区名称，格式为 `[Hemisphere]_[Region]`

**数值说明**：
- 投射强度（axon length，单位：微米）
- 可以是原始值或log转换后的值
- 0或NA表示无投射

### 1.3 Sheet 2: Metadata

**用途**：存储每个神经元的元信息

**结构示例**：

| Neuron_ID | Soma_Hemisphere | Soma_Region | Soma_Subregion | Neuron_Type | Total_Axon_Length | N_Branches |
|-----------|-----------------|-------------|----------------|-------------|-------------------|------------|
| Neuron_1  | Left            | Insula      | Iai            | IT          | 15000             | 45         |
| Neuron_2  | Right           | PFC         | Area46         | PT          | 23000             | 62         |
| Neuron_3  | Left            | Insula      | FI             | IT          | 18000             | 50         |

**必需列**：

| 列名 | 说明 | 允许值 |
|-----|------|--------|
| `Neuron_ID` | 神经元唯一ID | 任意字符串，需与Projection_Data匹配 |
| `Soma_Hemisphere` | 胞体所在半球 | `Left`, `Right` |
| `Soma_Region` | 胞体所在大区域 | `Insula`, `PFC`, `ACC`, `Parietal`, `Temporal`等 |
| `Soma_Subregion` | 胞体亚区 | 具体区域名，如`Iai`, `FI`, `Area46`等 |
| `Neuron_Type` | 神经元类型 | `IT`, `PT`, `CT`, `ITc`, `ITs`等 |
| `Total_Axon_Length` | 总轴突长度 | 数值（微米） |
| `N_Branches` | 分支数量 | 整数 |

**可选列**：
- `Layer`：胞体所在皮层层次
- `Injection_Site`：病毒注射位点
- `Animal_ID`：动物编号
- `Experiment_Date`：实验日期

### 1.4 Sheet 3: Region_Info

**用途**：存储各脑区的信息

**结构示例**：

| Region_Name | Hemisphere | Major_Division | Cytoarchitecture | Functional_System | Is_Insula |
|-------------|------------|----------------|------------------|-------------------|-----------|
| Left_Ig     | Left       | Posterior      | Granular         | Interoceptive     | TRUE      |
| Left_Iai    | Left       | Anterior       | Agranular        | Affective         | TRUE      |
| Left_FI     | Left       | Anterior       | Agranular        | Self-awareness    | TRUE      |
| Right_Ig    | Right      | Posterior      | Granular         | Interoceptive     | TRUE      |
| Left_ACC    | Left       | Medial         | Agranular        | Cognitive         | FALSE     |

**必需列**：

| 列名 | 说明 | 允许值 |
|-----|------|--------|
| `Region_Name` | 区域名称 | 需与Projection_Data的列名完全匹配 |
| `Hemisphere` | 半球 | `Left`, `Right` |
| `Major_Division` | 大体分区 | `Anterior`, `Posterior`, `Medial`, `Lateral`等 |
| `Cytoarchitecture` | 细胞构筑 | `Granular`, `Dysgranular`, `Agranular` |
| `Functional_System` | 功能系统 | `Interoceptive`, `Affective`, `Cognitive`等 |
| `Is_Insula` | 是否为脑岛 | `TRUE`, `FALSE` |

## 2. 数据填写规范

### 2.1 命名规范

**Neuron_ID**：
- 使用有意义的命名
- 推荐格式：`[Animal]_[Region]_[Number]`
- 示例：`M1_Insula_001`, `M2_PFC_045`

**Region_Name**：
- 格式：`[Hemisphere]_[Region]`
- 脑岛区域必须使用Evrard 2014命名：
  - `Left_FI`, `Right_FI`
  - `Left_Iai`, `Right_Iai`
  - `Left_Ial`, `Right_Ial`
  - `Left_Iam`, `Right_Iam`
  - `Left_Iapm`, `Right_Iapm`
  - `Left_Iapl`, `Right_Iapl`
  - `Left_Id`, `Right_Id`
  - `Left_Ig`, `Right_Ig`

### 2.2 数值规范

**投射强度**：
- 单位：微米（axon length）
- 范围：0到数十万
- 0表示无投射
- 建议使用原始值（代码会进行log转换）

**Total_Axon_Length**：
- 单位：微米
- 必须大于0

### 2.3 分类变量规范

**Soma_Hemisphere**：
- 必须完全一致：`Left`, `Right`
- 不要混用：`left`, `LEFT`, `L`等

**Neuron_Type**：
- 推荐值：`IT`, `PT`, `CT`, `ITc`, `ITs`
- 保持命名一致

**Cytoarchitecture**：
- 必须完全一致：`Granular`, `Dysgranular`, `Agranular`

## 3. 质量控制检查

### 3.1 检查清单

#### Sheet 1: Projection_Data

- [ ] Neuron_ID无重复
- [ ] Neuron_ID与Metadata匹配
- [ ] 列名与Region_Info匹配
- [ ] 数值无负数
- [ ] 无空值（可用0或NA填充）
- [ ] 至少包含一些脑岛相关列

#### Sheet 2: Metadata

- [ ] Neuron_ID无重复
- [ ] Neuron_ID与Projection_Data匹配
- [ ] Soma_Hemisphere值为`Left`或`Right`
- [ ] Soma_Region值一致
- [ ] Neuron_Type值一致
- [ ] Total_Axon_Length > 0
- [ ] N_Branches >= 0

#### Sheet 3: Region_Info

- [ ] Region_Name无重复
- [ ] Region_Name与Projection_Data列名匹配
- [ ] Hemisphere值为`Left`或`Right`
- [ ] Cytoarchitecture值为标准值
- [ ] Is_Insula为`TRUE`或`FALSE`
- [ ] 脑岛区域标记正确

### 3.2 R代码质量检查

```r
# ============================================================================
# 质量控制检查脚本
# ============================================================================

quality_check <- function(file_path) {
    cat("="*60, "\n")
    cat("  数据质量检查\n")
    cat("="*60, "\n\n")
    
    # 读取数据
    df_proj <- read_excel(file_path, sheet = "Projection_Data")
    df_meta <- read_excel(file_path, sheet = "Metadata")
    df_region <- read_excel(file_path, sheet = "Region_Info")
    
    errors <- 0
    warnings <- 0
    
    # 检查1: Neuron_ID匹配
    cat("📋 检查1: Neuron_ID匹配\n")
    proj_ids <- df_proj[[1]]
    meta_ids <- df_meta$Neuron_ID
    
    if (!all(proj_ids %in% meta_ids)) {
        cat("  ❌ 错误: Projection_Data中有Neuron_ID不在Metadata中\n")
        cat("     ", setdiff(proj_ids, meta_ids), "\n")
        errors <- errors + 1
    } else {
        cat("  ✅ Neuron_ID匹配正常\n")
    }
    
    # 检查2: Region_Name匹配
    cat("\n📋 检查2: Region_Name匹配\n")
    proj_regions <- colnames(df_proj)[-1]
    region_names <- df_region$Region_Name
    
    if (!all(proj_regions %in% region_names)) {
        cat("  ❌ 错误: Projection_Data中有列名不在Region_Info中\n")
        cat("     ", setdiff(proj_regions, region_names), "\n")
        errors <- errors + 1
    } else {
        cat("  ✅ Region_Name匹配正常\n")
    }
    
    # 检查3: 脑岛区域标记
    cat("\n📋 检查3: 脑岛区域标记\n")
    insula_regions <- df_region %>% 
        filter(Is_Insula == TRUE) %>% 
        pull(Region_Name)
    cat("  识别到", length(insula_regions), "个脑岛区域:\n")
    cat("  ", paste(insula_regions, collapse = ", "), "\n")
    
    # 检查4: 数值范围
    cat("\n📋 检查4: 数值范围\n")
    mat <- as.matrix(df_proj[, -1])
    if (any(mat < 0, na.rm = TRUE)) {
        cat("  ❌ 错误: 存在负值\n")
        errors <- errors + 1
    } else {
        cat("  ✅ 无负值\n")
    }
    
    cat("  投射强度范围: [", min(mat, na.rm = TRUE), ", ", 
        max(mat, na.rm = TRUE), "]\n")
    cat("  零值比例: ", sum(mat == 0, na.rm = TRUE) / length(mat) * 100, "%\n")
    
    # 检查5: 重复值
    cat("\n📋 检查5: 重复检查\n")
    if (any(duplicated(df_meta$Neuron_ID))) {
        cat("  ❌ 错误: Metadata中存在重复的Neuron_ID\n")
        errors <- errors + 1
    } else {
        cat("  ✅ 无重复Neuron_ID\n")
    }
    
    # 总结
    cat("\n" + "="*60, "\n")
    cat("  检查结果: ", errors, "个错误, ", warnings, "个警告\n")
    cat("="*60, "\n")
    
    return(list(errors = errors, warnings = warnings))
}

# 运行检查
quality_check("your_data.xlsx")
```

## 4. 常见问题排查

### 4.1 错误：Neuron_ID不匹配

**症状**：
```
❌ 错误: Projection_Data中有Neuron_ID不在Metadata中
```

**解决**：
1. 检查两个Sheet的Neuron_ID列名是否一致
2. 检查是否有拼写错误
3. 检查是否有空格

```r
# 查找差异
setdiff(df_proj[[1]], df_meta$Neuron_ID)
setdiff(df_meta$Neuron_ID, df_proj[[1]])
```

### 4.2 错误：Region_Name不匹配

**症状**：
```
❌ 错误: Projection_Data中有列名不在Region_Info中
```

**解决**：
1. 检查列名拼写
2. 检查是否有空格
3. 检查大小写

```r
# 查看不匹配的列名
setdiff(colnames(df_proj)[-1], df_region$Region_Name)
```

### 4.3 警告：零值比例过高

**症状**：
```
⚠️ 警告: 零值比例超过80%
```

**可能原因**：
1. 数据稀疏（正常）
2. 单位错误（如用了mm而不是μm）
3. 数据缺失

**解决**：
- 检查数据单位
- 确认0值是否合理

### 4.4 错误：负值存在

**症状**：
```
❌ 错误: 存在负值
```

**解决**：
- 检查数据源
- axon length不可能为负
- 重新检查数据
