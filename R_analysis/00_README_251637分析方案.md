## 🔬 脑岛分析要点

### 胞体位置: CR_Ial
- **C** = Contralateral (对侧) 或 **R** = Right (右)
- **R** = Rostral/Right
- **Ial** = Insula agranular anterior, lateral
- 属于前岛叶agranular区
- VENs密度较低

### 分析重点
1. **脑岛内连接**: Ial神经元投射到哪些脑岛亚区？
2. **脑岛外投射**: 主要投射到哪些皮层/皮层下区域？
3. **侧向性**: 同侧vs对侧投射比例
4. **聚类特异性**: 不同形态学聚类的投射特征
