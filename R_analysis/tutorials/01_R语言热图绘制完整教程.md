## 1. 文献热图分析

### 1.1 Figure 2A - 主投射热图

**用途**：展示32个神经元亚型在各脑区的投射强度

**关键元素**：
- **行**：32个神经元亚型（按层次聚类排列）
- **列**：目标脑区（分为Ipsilateral和Contralateral两组）
- **数值**：log(axon length)，表示投射强度
- **顶部注释条**：
  - Type分类（ITc/ITs/CT/PT/ITi）- 5种颜色
  - Soma区域分布
  - 神经元数量（条形图）
  - 总轴突长度（条形图）
- **聚类树状图**：显示亚型间的相似性关系

**为什么这样设计**：
- 层次聚类揭示投射模式的相似性
- 注释条提供多维度的生物学信息
- log转换压缩动态范围，突出弱投射

### 1.2 Figure 4 - Intra-PFC连接热图

**用途**：展示PFC内部44个子域之间的连接模式

**关键元素**：
- **矩阵热图**：子域间的投射强度
- **网络图**：基于阈值过滤后的连接
- **模块化高亮**：dlPFC（青色）和vlPFC（橙色）

**为什么这样设计**：
- 矩阵展示全连接模式
- 网络图突出关键连接
- 颜色编码区分功能模块

### 1.3 Figure S3 - Arborization模式热图

**用途**：展示轴突末梢的树状分支模式

**关键元素**：
- **层分布热图**：轴突在各皮层的分布
- **聚类结果**：6种arborization类型

**为什么这样设计**：
- 层特异性反映功能特性
- 聚类识别结构模式

## 2. 为什么选择这些热图

### 2.1 生物学意义

| 热图类型 | 生物学问题 | 分析方法 |
|---------|-----------|---------|
| 投射强度热图 | 神经元投射到哪里？ | 层次聚类 + 注释 |
| 连接矩阵热图 | 区域间如何连接？ | 相关性 + 网络分析 |
| 层分布热图 | 轴突如何分布？ | 密度聚类 |

### 2.2 可视化原则

1. **数据-墨水比**：最大化数据信息，最小化装饰
2. **层次结构**：从总体到细节
3. **多维度整合**：在一个图中展示多种信息
4. **可比较性**：颜色、尺度一致

## 4. ComplexHeatmap深度解析

### 4.1 核心函数

```r
library(ComplexHeatmap)

# 基础热图
Heatmap(matrix, 
        name = "legend_title",
        col = color_palette,
        cluster_rows = TRUE,
        cluster_columns = TRUE)
```

### 4.2 注释系统

```r
# 顶部注释
ha_top <- HeatmapAnnotation(
    Type = metadata$Type,
    Soma = metadata$Soma_Region,
    col = list(
        Type = c("ITc" = "#4DAF4A", "ITs" = "#984EA3"),
        Soma = c("Area45" = "red", "Area46" = "blue")
    )
)

# 应用到热图
Heatmap(matrix, top_annotation = ha_top)
```

### 4.3 聚类控制

```r
# 自定义聚类
dist_matrix <- dist(matrix, method = "euclidean")
hc <- hclust(dist_matrix, method = "ward.D2")

Heatmap(matrix, cluster_rows = hc)
```

## 5. 脑岛分析的特殊考虑

### 5.1 脑岛分区体系（Evrard 2012/2014）

**细胞构筑分区**：
- **Granular (Ig)**：有清晰的layer IV，位于后岛叶
- **Dysgranular (Id)**：layer IV不规则，位于中岛叶
- **Agranular (Ia)**：无layer IV，位于前岛叶

**亚区命名**：
- **Iai**：insula agranular anterior, intermediate（含VENs）
- **Ial**：insula agranular anterior, lateral
- **Iam**：insula agranular anterior, medial
- **Iapm**：insula agranular posterior, medial
- **Iapl**：insula agranular posterior, lateral
- **FI**：frontoinsular cortex（前岛叶最前端，含大量VENs）

### 5.2 左右不对称性

脑岛存在明显的左右不对称：
- 右岛叶：更侧重自主神经和情绪处理
- 左岛叶：更侧重语言和认知功能

### 5.3 投射特征

根据Evrard 2019综述：
- 后岛叶（granular）：接收初级内感受输入
- 中岛叶（dysgranular）：多模态整合
- 前岛叶（agranular）：高级整合和输出
