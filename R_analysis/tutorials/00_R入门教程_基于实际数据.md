R是一种专门用于**统计计算**和**数据可视化**的编程语言。在神经科学研究中，R是绘制 publication-quality 图形的首选工具。

```r
# ========== 变量赋值 ==========
```

## 2. 理解你的数据结构

### 2.1 数据概览

你的数据包含 **212个神经元**，来自 **右岛叶Ial区 (CR_Ial)**：

| 文件 | 内容 | 用途 |
|-----|------|------|
| `251637_results.xlsx` | 完整投射数据 | 主要分析 |
| `251637_results_clustered_k9_spearman_penalty.xlsx` | k=9聚类结果 | 形态学分组 |
| `251637_laterality_projections.xlsx` | 侧向性数据 | 左右对比 |

### 2.2 关键Sheet说明

**Summary Sheet** (212行 × 18列)：
```
NeuronID    : 神经元编号 (001.swc, 002.swc...)
Neuron_Type : 神经元类型 (ITs, ITi, CT, ITc)
Soma_Region : 胞体区域 (CR_Ial = 右岛叶Ial)
Soma_Side   : 左右 (R = Right)
Total_Length: 总轴突长度
Terminal_Count: 终端数量
N_Ipsilateral : 同侧投射区域数
Laterality_Index: 侧向性指数
```

**Projection_Length Sheet** (212行 × 263列)：
```
NeuronID, Neuron_Type + 260个脑区
例如: CL_AI, CL_AIP, SR_ac, SR_ic...
```

**Projection_Strength Sheet**：
- 与Length相同结构，但数值是强度（可能是log转换）

**Soma_Hierarchy Sheet**：
```
L1: Frontal
L2: OFC
L3: caudal_OFC
L4: OFa-p
L5: lat_Ia
L6: Ial
```

### 2.3 神经元类型说明

| 类型 | 全称 | 投射特征 |
|-----|------|---------|
| ITs | Intratelencephalic, striatum-projecting | 投射到纹状体 |
| ITi | Intratelencephalic, ipsilateral | 同侧皮层投射 |
| ITc | Intratelencephalic, contralateral | 对侧皮层投射 |
| CT  | Corticothalamic | 皮层-丘脑投射 |

## 5. 热图绘制入门

### 5.1 使用pheatmap（简单热图）

```r
library(pheatmap)

# ========== 准备数据 ==========
# 提取投射矩阵（去掉NeuronID和Neuron_Type列）
proj_matrix <- df_proj_length %>%
    select(-NeuronID, -Neuron_Type) %>%
    as.matrix()

# 设置行名
rownames(proj_matrix) <- df_proj_length$NeuronID

# 取log1p转换（处理0值）
proj_matrix_log <- log1p(proj_matrix)

# ========== 简单热图 ==========
pheatmap(proj_matrix_log[1:20, 1:30],  # 先画前20个神经元和前30个脑区
         cluster_rows = TRUE,
         cluster_cols = TRUE,
         scale = "none",
         main = "Projection Length Heatmap (log)")

# ========== 添加注释 ==========
# 创建行注释
row_annotation <- data.frame(
    Type = df_proj_length$Neuron_Type[1:20]
)
rownames(row_annotation) <- rownames(proj_matrix_log)[1:20]

# 带注释的热图
pheatmap(proj_matrix_log[1:20, 1:30],
         cluster_rows = TRUE,
         cluster_cols = TRUE,
         annotation_row = row_annotation,
         main = "Projection Heatmap with Neuron Type")
```

### 5.2 使用ComplexHeatmap（高级热图）

```r
library(ComplexHeatmap)
library(circlize)

# ========== 颜色方案 ==========
# 神经元类型颜色
type_colors <- c(
    "ITs" = "#4DAF4A",
    "ITi" = "#984EA3",
    "ITc" = "#377EB8",
    "CT" = "#FF7F00"
)

# 投射强度颜色（蓝色渐变）
col_fun <- colorRamp2(
    c(0, 2, 4, 6, 8),
    c("#FFFFFF", "#C6DBEF", "#6BAED6", "#2171B5", "#08306B")
)

# ========== 创建顶部注释 ==========
ha_top <- HeatmapAnnotation(
    Neuron_Type = df_proj_length$Neuron_Type[1:50],
    col = list(Neuron_Type = type_colors),
    annotation_name_side = "left"
)

# ========== 创建热图 ==========
ht <- Heatmap(
    proj_matrix_log[1:50, 1:50],
    name = "Log Length",
    col = col_fun,
    cluster_rows = TRUE,
    cluster_columns = TRUE,
    top_annotation = ha_top,
    row_names_gp = gpar(fontsize = 6),
    column_names_gp = gpar(fontsize = 6),
    column_names_rot = 45,
    show_row_names = FALSE
)

# 绘制
draw(ht)

# 保存为PDF
pdf("heatmap_complex.pdf", width = 12, height = 10)
draw(ht)
dev.off()
```

### 6.1 完整的数据读取和预处理

```r
# ========== 完整的数据读取脚本 ==========

library(readxl)
library(dplyr)
library(ComplexHeatmap)
library(circlize)

cat("Reading data...\n")

df_summary <- read_excel(file_results, sheet = "Summary")
df_hierarchy <- read_excel(file_results, sheet = "Soma_Hierarchy")
df_proj_length <- read_excel(file_results, sheet = "Projection_Length")
df_proj_strength <- read_excel(file_results, sheet = "Projection_Strength")
df_laterality <- read_excel(file_results, sheet = "Laterality")

cat("Data loaded successfully!\n")
cat("Neurons:", nrow(df_summary), "\n")
cat("Brain regions:", ncol(df_proj_length) - 2, "\n")

# 4. 合并数据
df_full <- df_summary %>%
    left_join(df_hierarchy, by = "NeuronID") %>%
    left_join(df_clustered %>% select(NeuronID, Morph_Cluster), by = "NeuronID")

# 5. 准备投射矩阵
proj_matrix <- df_proj_length %>%
    select(-NeuronID, -Neuron_Type) %>%
    as.matrix()
rownames(proj_matrix) <- df_proj_length$NeuronID

# 6. 筛选有投射的脑区（去掉全0列）
col_sums <- colSums(proj_matrix)
proj_matrix_filtered <- proj_matrix[, col_sums > 0]

cat("Regions with projections:", ncol(proj_matrix_filtered), "\n")
```

```r
# ========== 按k=9聚类分组的热图 ==========

# 将聚类信息添加到投射数据
df_proj_with_cluster <- df_proj_length %>%
```
