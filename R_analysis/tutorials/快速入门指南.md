## 第二步：准备你的数据

### Excel文件结构

创建名为 `neuron_projection_data.xlsx` 的文件，包含3个Sheet：

#### Sheet 1: Projection_Data
| Subtype | ACC | MCC | dlPFC | vlPFC | ... |
|---------|-----|-----|-------|-------|-----|
| Sub1    | 2.5 | 1.2 | 3.4   | 0.8   | ... |
| Sub2    | 0.3 | 2.1 | 1.5   | 3.2   | ... |
| Sub3    | 1.8 | 0.9 | 2.2   | 1.1   | ... |

**注意：**
- 第一列是亚型名称
- 后面的列是各个脑区的投射强度（log轴突长度）
- 数值可以是原始长度，代码会进行log转换

#### Sheet 2: Metadata
| Subtype | Type | Soma_Region | Total_Axon_Length | N_Neurons |
|---------|------|-------------|-------------------|-----------|
| Sub1    | ITc  | Area_45     | 15000             | 12        |
| Sub2    | ITs  | Area_46     | 23000             | 18        |
| Sub3    | PT   | Area_8A     | 45000             | 25        |

**字段说明：**
- `Subtype`: 亚型名称（必须与Projection_Data对应）
- `Type`: 主要类型（ITc/ITs/CT/PT/ITi）
- `Soma_Region`: 胞体所在区域
- `Total_Axon_Length`: 总轴突长度
- `N_Neurons`: 该亚型的神经元数量

#### Sheet 3: Region_Info
| Region | Hemisphere | Category |
|--------|------------|----------|
| ACC    | Ipsilateral| Cortical |
| MCC    | Ipsilateral| Cortical |
| Thal   | Ipsilateral| Subcortical|

**字段说明：**
- `Region`: 脑区名称（必须与Projection_Data的列名对应）
- `Hemisphere`: 同侧/对侧 (Ipsilateral/Contralateral)
- `Category`: 脑区类别

## 常见问题速查

### Q: 报错 "找不到文件"
**解决：** 检查文件路径是否正确，使用绝对路径：
```r
file_path <- "C:/Users/你的名字/Documents/neuron_projection_data.xlsx"
```

### Q: 报错 "没有read_excel函数"
**解决：** 安装readxl包
```r
install.packages("readxl")
```

### Q: 热图显示不全
**解决：** 调整PDF尺寸
```r
pdf("output.pdf", width = 16, height = 12)  # 增大尺寸
draw(ht)
dev.off()
```

### Q: 颜色不满意
**解决：** 修改颜色方案
```r
col_fun <- colorRamp2(
  c(0, 2, 5), 
  c("white", "yellow", "red")  # 修改这里
)
```
